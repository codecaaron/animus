# @animus-ui/properties

CSS property classification data for the Animus extraction pipeline. Provides the unitless property set, shorthand-to-longhand mappings, and CSS type re-exports.

This is a transitive dependency of `@animus-ui/system`. You don't need to install it directly.

## What It Provides

- **Unitless properties**: Set of CSS properties that don't need `px` suffix (e.g., `lineHeight`, `opacity`, `zIndex`)
- **Shorthand list**: Maps shorthand properties to their longhand expansions for cascade ordering
- **Type re-exports**: Shared CSS type definitions
