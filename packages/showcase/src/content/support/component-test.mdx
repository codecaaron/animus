## Callout Variants

<Callout variant="info" title="Cross-reference">
  The cascade contract is documented in the Architecture section. Each builder
  method maps to a specific `@layer` — see the chain visualization above.
</Callout>

<Callout variant="tip" title="Best practice">
  Define your color scheme as a variant that rebinds `--color-scheme-*` weights.
  Style variants then reference `scheme.500`, `scheme.400` etc. — the component
  never knows which palette is active.
</Callout>

<Callout variant="warn" title="Breaking in v2.3">
  The `.states()` map signature changed from state names to CSS selectors as
  keys. See the migration guide for details on updating existing components.
</Callout>

<Callout variant="danger" title="Do not">
  Never add React resolve aliases to `vite.config.ts`. They silently break the
  extraction transform pipeline. If you need to alias a package, use the
  `resolve.alias` field in the Vite config with extreme caution.
</Callout>

<Callout variant="deprecated" title="Deprecated · v2.x">
  The `.system()` group binding API is deprecated. Use `.includes()` for cosmetic grouping or bind props directly via `.styles()`. Migration: replace `.system({ space: true })` with `.includes(space)`.
</Callout>

### Collapsible

<SyntaxBlock title="full-system-setup.ts" language="tsx" showLineNumbers collapsible>{`import { createSystem, createTheme, createTransform } from '@animus-ui/system';
import { space, typography, color, border, shadows } from '@animus-ui/system/groups';

const fluid = createTransform('fluid', (value) => {
const [min, max] = String(value).split('-').map(Number);
const minRem = min / 16;
const maxRem = max / 16;
const slope = ((max - min) / 880) _ 100;
const intercept = minRem - (slope / 100) _ (480 / 16);
return \`clamp(\${minRem}rem, \${intercept}rem + \${slope}vw, \${maxRem}rem)\`;
});

export const tokens = createTheme()
  .addBreakpoints({ sm: 768, md: 1024, lg: 1200 })
  .addColors({
    gray: { 50: '#fafafa', 900: '#111' },
    fire: { 500: '#FF2800', 700: '#C1121F' },
  })
  .addScale({
    name: 'space',
    values: { 4: '0.25rem', 8: '0.5rem', 16: '1rem' },
  })
  .build();

export const { system: ds } = createSystem()
  .addGroup('surface', { ...color, ...border, ...shadows })
  .addGroup('text', { ...typography })
  .addGroup('space', { ...space })
  .build();`}</SyntaxBlock>
