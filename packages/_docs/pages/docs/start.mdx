# Introduction

## Installation

```sh
yarn add @animus-ui/core @emotion/styled @emotion/react
```

## API

1. Create your base props and create your component builder.
2. Use builder to create end to end presentational elements.

## Configure

```tsx
const animus = createAnimus()
  .addGroup('space', {
    m: { property: 'margin' },
    p: { property: 'padding' },
  })
  .build();
```

## Build

```tsx
const FlexBox = animus
  .styles({ display: 'flex' })
  .states({
    inline: {
      display: 'inline-flex',
    },
  })
  .groups({
    space: true,
  })
  .props({
    gap: {
      property: 'gap',
      scale: {
        md: 12,
        lg: 24,
      },
    },
  })
  .asElement('div');

<Box p="2rem" gap="md" inline />;
```

## Theming

Structured theme builder.

```tsx
const theme = createTheme({
  breakpoints: {
    xs: 480,
    sm: 768,
    md: 1024,
    lg: 1200,
    xl: 1440,
  },
  fonts: {
    body: '"Open Sans", sans-serif',
    heading: '"Raleway", serif',
  },
  fontSizes: {
    12: '0.75rem',
    14: '0.875rem',
    16: '1rem',
    32: '2rem',
    44: '2.75rem',
    64: '4rem',
  },
})
  .addColors({
    black: '#000000',
    white: '#ffffff',
  })
  .createColorModes('light', {
    light: {
      text: 'black',
      background: 'white',
    },
    dark: {
      text: 'white',
      background: 'black',
    },
  })
  .build();
```
