Refined

Callout variants

Five severity levels — including tip and deprecation, which were missing from the original.

Info

Extraction runs at build time. No runtime JS is shipped to the client.

Tip

Use hash: "short" in development for readable class names while debugging.

Warning

Dynamic expressions inside css() calls cannot be extracted. Use variant maps instead.

Error

Circular token references will cause the compiler to throw during the resolve phase.

Deprecated · v2.x

The styled() API is deprecated. Migrate to css() + classMap before v3.0.

Migration checklist

Step-by-step migration path with progress dots — a common gap in compiler docs.

Install the compiler plugin
Add @acme/compiler and configure your bundler integration.
Define your theme config
Move design tokens from JS objects to acme.config.ts.
Replace styled() with css()
Use the codemod: npx @acme/codemod styled-to-css
Remove runtime dependency
Uninstall the runtime package once all components are migrated.
